# GitHub Pages 上線備忘錄

此網站包是靜態網站，包含 HTML / CSS / JS / images。

## 建議 repository 名稱

`ann-legal-tax-ai-lab`

## GitHub Pages 設定

1. 建立新的 repository。
2. 將本資料夾內所有檔案上傳到 repository 根目錄。
3. 到 Settings → Pages。
4. Source 選 `Deploy from a branch`。
5. Branch 選 `main`，Folder 選 `/root`。
6. 儲存後等待 GitHub 產生網址。
