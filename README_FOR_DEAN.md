# Ann 法稅 AI Lab WebPack v0.1｜給 Dean

## 最重要修正
本包是「真圖素材版 UI / CSS 原型」，不是 AI 生成圖網站。所有人物圖只能使用 Ann 提供並確認過的原始照片素材。

## 不可以做
- 不可以重新生成 Ann 的臉
- 不可以重新生成所長的臉
- 不可以重新生成合照
- 不可以改姿勢、改服裝、改背景
- 不可以在圖片裡生成網站名稱、logo、律所名稱或職稱
- 不可以用 AI 自己猜任遠律所資訊

## 可以做
- 裁切
- 調整比例
- 加 dark overlay / gradient
- 加圓角
- 加陰影
- HTML / CSS 排版
- RWD 裁切

## 五頁
- index.html
- about.html
- topics.html
- content.html
- contact.html

## 接口保留
Content 頁已保留 Podcast / YouTube / Article / Reels / Clips 等 link slot，未來只要替換 href 即可。
